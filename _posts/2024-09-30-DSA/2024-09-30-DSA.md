## 滑动窗口
> 给定一个长度为n（n<=10^6）的数组。有一个大小为k的滑动窗口从数组的最左端移动到最右端。你可以看到窗口中的k个数字。窗口每次向右滑动一个数字的距离。你的任务是得到滑动窗口在每个位置时的最大值和最小值。

>输入包括两行。<br>
第一行包括n和k，分别表示数组的长度和窗口的大小。<br>
第二行包括n个数字。

>输出包括两行。<br>
第一行包括窗口从左至右移动的每个位置的最小值。<br>
第二行包括窗口从左至右移动的每个位置的最大值。

> 样例输入<br>
8 3<br>
1 3 -1 -3 5 3 6 7

> 样例输出<br>
-1 -3 -3 -3 3 3<br>
3 3 5 5 6 7

### 题解
考虑最大值，窗口每右移一个元素（即每次循环），向min_index（最小值可能值的vector）中放入新增元素的索引，并删去min_index中所有比该元素小的元素，删去与新增元素index之差大于窗口长度的元素（每次只可能是第一个），保证min_index的首个元素总是当前窗口中的最大值，在循环次数i>=窗口长度时（完整出现窗口时）pop_front到min_ans中，最后一起输出。最小值同理。

### 代码实现
```cpp
int main() {
    int n, k;
    cin >> n >> k;
    vector<int> num{};
    deque<int> min_idx{}; //possible idx
    deque<int> max_idx{};
    vector<int> min_ans{}; //answer
    vector<int> max_ans{};
    for (int i = 0; i < n; i++) {
        int temp;
        cin >> temp;
        num.push_back(temp); //read data into num
        //删掉之前比当前大的索引
        while (!min_idx.empty() && num[min_idx.back()] >= num[i]) { 
            min_idx.pop_back();
        }
        //放入当前索引
        min_idx.push_back(i); 
        while (!max_idx.empty() && num[max_idx.back()] <= num[i]) {
            max_idx.pop_back();
        }
        max_idx.push_back(i);
        //删掉移出窗口的索引
        if (i - min_idx.front() >= k) {
            min_idx.pop_front();
        } 
        if (i - max_idx.front() >= k) {
            max_idx.pop_front();
        } 
        //完整窗口时放入答案向量
        if (i >= k - 1) {
            min_ans.push_back(num[min_idx.front()]);
        }
        if (i >= k - 1) {
            max_ans.push_back(num[max_idx.front()]);
        }
        
    } 
    for (auto it = min_ans.begin(); it != min_ans.end(); it++) {
        cout << *it <<" ";
    }
    cout << endl;
    for (auto it = max_ans.begin(); it != max_ans.end(); it++) {
        if (it == max_ans.begin()) cout << *it;
        else cout << " " << *it ;
    }
}
```

## 堆栈基本操作
>依次读入序列元素1,2,...,n进栈，每进一个元素，机器可要求下一个元素进栈或弹栈，如此进行。给定一个输入序列，判断栈空时弹出的元素构成的序列是否可能等于给定的序列，如果是则输出栈的操作过程，否则输出“NO”。

>输入
输入分两行<br>
第一行为n的值（即序列元素个数）<br>
第二行为给定的输入序列（序列元素均为整型）

>输出
如果输入序列能够由题目规定的操作得到，则输出对栈的操作过程<br>
否则直接输出“NO”

>样例输入<br>
7<br>
4 5 3 6 2 7 1

>样例输出<br>
PUSH 1<br>PUSH 2<br>PUSH 3<br>PUSH 4<br>POP 4<br>PUSH 5<br>POP 5<br>POP 3<br>PUSH 6<br>POP 6<br>POP 2<br>PUSH 7<br>POP 7<br>POP 1

>提示：给定序列中有可能有不在1...n之间的数字

### 题解
每遇到一个数，把1...n中<=该数的数都压栈，弹出该数。输入序列不能构成当且仅当压栈后栈顶元素与序列元素不等（包括栈为空的情况），存在1...n之外的元素当且仅当最后栈中有元素剩余。

### 代码实现
*这是我看题之后写下的代码，在检查和输出部分有冗余，代码风格不好，但复制黏贴很符合考试情境，应该在check中加一个operation向量来记录操作，符合条件时直接输出operation，而不是再次计算。*
```cpp
#include <iostream>
#include <deque>
#include <vector>
using namespace std;
//检查该序列是否能构成
bool check(deque<int> seq) {
    int k = 1;
    vector<int> stac; //栈
    //输入序列不为空
    while (!seq.empty()) {
        int first = seq.front();
        //比它小的都压栈
        for (; k <= first; k++) {
            stac.push_back(k);
        }
        //栈顶元素等于序列首部
        if (!stac.empty() && stac.back() == seq.front()) {
            stac.pop_back(); //弹栈
            seq.pop_front(); //移出序列
        } else {
            return false;
        }
    }
    return true;
}
//输出构成方式
void out(deque<int>& seq) {
    int k = 1;
    vector<int> stac;
    while (!seq.empty()) {
        int first = seq.front();
        for (; k <= first; k++) {
            stac.push_back(k);
            cout << "PUSH " << k << endl;
        }
        if (!stac.empty()) {
            cout << "POP " << stac.back() << endl;
            stac.pop_back();
            seq.pop_front();
        }
    }
}

int main() {
    int n;
    cin >> n;
    deque<int> seq;
    while (n--) {
        int temp;
        cin >> temp;
        seq.push_back(temp);
    }
    if (check(seq)) {
        out(seq);
    } else {
        cout << "NO" << endl;
    }
    return 0;
}
```
*这是GPT-4o-Mini提供的实现，不是通过将小于等于当前元素作为压栈条件，而是栈顶元素不等于当前元素时就压栈，inputIndex是1...n，outputIndex是输入序列索引，看着没什么问题，但openjudge上WA。*
```cpp
#include <iostream>
#include <stack>
#include <vector>
using namespace std;

vector<string> checkStackSequence(int n, vector<int>& outputSequence) {
    stack<int> S;
    int inputIndex = 1;
    int outputIndex = 0;
    vector<string> operations;
    while (outputIndex < n) {
        // 如果栈为空或栈顶元素不符合出栈要求
        if (S.empty() || S.top() != outputSequence[outputIndex]) {
            if (inputIndex > n) {
                break; // 没有更多元素可进栈
            }
            // 进栈操作
            S.push(inputIndex);
            operations.push_back("Push " + to_string(inputIndex));
            inputIndex++;
        } else {
            // 出栈操作
            int poppedElement = S.top();
            S.pop();
            operations.push_back("Pop " + to_string(poppedElement));
            outputIndex++;
        }
    }
    // 检查是否成功完成出栈序列
    if (outputIndex == n) {
        return operations;
    } else {
        return {"NO"};
    }
}

int main() {
    int n;
    cin >> n;
    vector<int> outputSequence(n);
    for (int i = 0; i < n; ++i) {
        cin >> outputSequence[i];
    }
    vector<string> result = checkStackSequence(n, outputSequence);
    for (const string& operation : result) {
        cout << operation << endl;
    }
    return 0;
}
```

## 中缀表达式的值
>人们熟悉的四则运算表达式称为中缀表达式，例如(23+34*45/(5+6+7))。在程序设计语言中，可以利用堆栈的方法把中缀表达式转换成保值的后缀表达式（又称逆波兰表示法），并最终变为计算机可以直接执行的指令，得到表达式的值。给定一个中缀表达式，编写程序，利用堆栈的方法，计算表达式的值。

>输入<br>
第一行为测试数据的组数N<br>
接下来的N行，每行是一个中缀表达式。表达式中只含数字、四则运算符和圆括号，操作数都是正整数，数和运算符、括号之间没有空格。中缀表达式的字符串长度不超过600。

>输出<br>
对每一组测试数据输出一行，为表达式的值

>样例输入<br>
3<br>
3+5\*8 <br>
(3+5)\*8 <br>
(23+34*45/(5+6+7))

>样例输出<br>
43<br>
64<br>
108

### 题解
计算概论中有过将中缀表达式转化成后缀表达式的题，加上数算课本有过类似的例题，很自然想到先将中缀表达式转化为后缀表达式，在对后缀表达式求值。<br>
中缀表达式转化为后缀表达式的步骤如下：
- 操作数：直接放入后缀表达式
- 开括号：压栈
- 闭括号：弹栈直至开括号
- 操作符：栈非空&&栈顶不是开括号&&栈顶运算符优先级>=该运算符，反复弹出；该运算符入栈
- 遍历中缀表达式后，栈内剩余元素依次弹出

后缀表达式求值步骤如下：
- 操作数：压栈
- 操作符：将栈顶两个操作数弹出，运算后将结果压栈
- 最后栈内应当只有一个元素，弹出即为结果

实现时遇到的一个问题是：用字符串存储中缀和后缀表达式，将其中的一个操作数的不同数位视作整体，而不同操作数之间加上空格分离。

### 代码实现
```cpp
#include <iostream>
#include <deque>
#include <vector>
#include <string>
using namespace std;


bool isDigit(char& c) {
    return (c <= '9' && c >= '0');
}
bool isOperator(char& c) {
    return (c == '+' || c == '-' || c == '/' || c == '*');
}
int precedence(char& c) {
    if (c == '+' || c == '-') return 1;
    else return 2;
}
string mid2rear(string& midExpr) {
    string rearExpr;
    vector<char> stac;
    for (char c : midExpr) {
        if (isDigit(c)) rearExpr += c;
        else if (c == '(') stac.push_back(c);
        else if (c == ')') {
            if (!stac.empty()) {
                while (stac.back() != '(') {
                    char temp = stac.back();
                    stac.pop_back();
                    rearExpr += temp;
                }
                stac.pop_back(); // '('
            }
        }
        else if (isOperator(c)) {
            rearExpr += ' ';
            while (!stac.empty() && stac.back() != '(' && precedence(stac.back()) >= precedence(c)) {
                char temp = stac.back();
                stac.pop_back();
                rearExpr += temp;
            }
            stac.push_back(c);
        }
    }
    while (!stac.empty()) {
        char temp = stac.back();
        stac.pop_back();
        rearExpr += temp;
    }
    return rearExpr;
}

int calc(string rearExpr) {
    vector<int> result;
    bool lastIsDig = false;
    for (char c : rearExpr) {
        if (isDigit(c)) {
            if (!lastIsDig) {
                result.push_back((int)(c - '0'));
            } else {
                int temp = result.back();
                temp = 10*temp + (int)(c - '0');
                result.pop_back();
                result.push_back(temp);
            }
            lastIsDig = true;
        } else {
            lastIsDig = false;
            if (c == ' ') continue; 
            else if (c == '+') {
                int temp1 = result.back();
                result.pop_back();
                int temp2 = result.back();
                result.pop_back();
                result.push_back(temp1 + temp2);
            }
            else if (c == '-') {
                int temp1 = result.back();
                result.pop_back();
                int temp2 = result.back();
                result.pop_back();
                
                result.push_back(temp2 - temp1);
            }
            else if (c == '*') {
                int temp1 = result.back();
                result.pop_back();
                int temp2 = result.back();
                result.pop_back();
                result.push_back(temp1 * temp2);
            }
            else if (c == '/') {
                int temp1 = result.back();
                result.pop_back();
                int temp2 = result.back();
                result.pop_back();
                result.push_back(temp2 / temp1);
            }
        }
        // for(auto i = result.begin();i!=result.end();i++)
        // cout<<*i<<" ";
    }
    return result.back();
}

int main() {
    int n;
    cin >> n;
    while (n--) {
        string midExpr;
        cin >> midExpr;
        if (n > 1)
        cout << calc(mid2rear(midExpr)) << " " << endl;
        else
        cout << calc(mid2rear(midExpr)) << endl;
    }
}
```
