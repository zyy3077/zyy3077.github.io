# 第五章 二叉树
题目详见：[http://dsa.openjudge.cn/2024dsachapter5/](http://dsa.openjudge.cn/2024dsachapter5/)

## Huffman编码树
构造一个具有n个外部节点的扩充二叉树，每个外部节点Ki有一个Wi对应，作为该外部节点的权。使得这个扩充二叉树的叶节点带权外部路径长度总和最小：

                                     Min( W1 * L1 + W2 * L2 + W3 * L3 + … + Wn * Ln)
Wi:每个节点的权值。<br>
Li:根节点到第i个外部叶子节点的距离。<br>
编程计算最小外部路径长度总和。


输入<br>
第一行输入一个整数t，代表测试数据的组数。<br>
对于每组测试数据，第一行输入一个整数n，外部节点的个数。第二行输入n个整数，代表各个外部节点的权值。<br>
2<=n<=100

输出<br>
输出最小外部路径长度总和。

样例输入<br>
2<br>
3<br>
1 2 3<br>
4<br>
1 1 3 5

样例输出<br>
9<br>
17

### 题解
构造Huffman二叉树的过程：每次选两个值最小的节点合并，合并后节点的值为二者之和，叶子节点到根节点距离*叶子节点值之和为带权外部路径长度总和。<br>
用stl中的优先队列模拟最小堆，每次合并pop前两个元素并push二者之和，累加pop出的元素值即可，因为在第n层的叶子节点自身被累加或者它与另一个值的和作为节点被累加，一共n次，即累加了它的带权路径长度。<br>
priority_queue的基本用法如下:

                                     std::priority_queue<int> max_pq; // 默认是最大优先队列
                                     std::priority_queue<int, std::vector<int>, std::greater<int>> min_pq; // 最小优先队列
Wi:每个节点的权值。<br>
Li:根节点到第i个外部叶子节点的距离<br>
编程计算最小外部路径长度总和


### 代码实现
```cpp
#include<iostream>
#include<queue>
#include<vector>
using namespace std;

int main()
{
    int t;
    cin >> t;
    while (t--) {
        int n, ans = 0;
        cin >> n;
        priority_queue<int,vector<int>, greater<int>>q;
        for(int i = 0; i < n; i ++) {
            int x;
            cin >> x;
            q.push(x);
        }
        while(q.size() != 1)
        {
            int t1,t2;
            t1 = q.top();q.pop();
            t2 = q.top();q.pop();
            ans += t1 + t2; //以自身or和的形式被累加n次，即乘以它的深度n
            q.push(t1 + t2);
        }	
        cout << ans << endl;
    }
	return 0;
}
```
