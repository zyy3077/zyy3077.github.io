第五章 二叉树

题目详见：http://dsa.openjudge.cn/2024dsachapter5/

基本概念

  • 满二叉树：所有节点的度数为2或0的二叉树，即左右子树均非空或均为空
  • 完全二叉树：所有节点尽量由左至右、由上至下排列，即最多只有最下面两层的节点度数可以小于2，并且最下面一层的节点都集中在该层最左边的连续位置上
  • 二叉搜索树：对于任意的节点，左子节点的值比它小，右子节点的值比它大
  • 堆：完全二叉树，且对于任意的节点，子节点的值均比它小（最大堆）或均比它大（最小堆），优先队列的一种实现方式
  • Huffman树：满二叉树，Huffman编码为前缀编码，\(k\)叉Huffman书构造时要根据数学性质补充值为0的节点

\(k\)叉树基本性质

  • 总边数\(e = \sum_{i} d_i = k \cdot n_k + … + 1 \cdot n_1 + 0 \cdot n_0 \)
  • 总边数\(e = n - 1 = n_0 + n_1 + … + n_k - 1\)

Huffman编码树

构造一个具有n个外部节点的扩充二叉树，每个外部节点Ki有一个Wi对应，作为该外部节点的权。使得这个扩充二叉树的叶节点带权外部路径长度总和最小：

Min( W1 * L1 + W2 * L2 + W3 * L3 + … + Wn * Ln)
  • Wi:每个节点的权值。
  • Li:根节点到第i个外部叶子节点的距离。
  • 编程计算最小外部路径长度总和。

输入：

  • 第一行输入一个整数t，代表测试数据的组数。
  • 对于每组测试数据，第一行输入一个整数n，外部节点的个数。第二行输入n个整数，代表各个外部节点的权值。
  • 2<=n<=100

输出：

  • 输出最小外部路径长度总和。

样例输入：

2
3
1 2 3
4
1 1 3 5

样例输出：

9
17

题解

构造Huffman二叉树的过程：每次选两个值最小的节点合并，合并后节点的值为二者之和，叶子节点到根节点距离*叶子节点值之和为带权外部路径长度总和。
用stl中的优先队列模拟最小堆，每次合并pop前两个元素并push二者之和，累加pop出的元素值即可，因为在第n层的叶子节点自身被累加或者它与另一个值的和作为节点被累加，一共n次，即累加了它的带权路径长度。
priority_queue的基本用法如下:

std::priority_queue<int> max_pq; // 默认是最大优先队列
std::priority_queue<int, 
            std::vector<int>, 
                    std::greater<int>> min_pq; // 最小优先队列

代码实现

//Huffman编码树
#include<iostream>
#include<queue>
#include<vector>
using namespace std;

int main()
{
    int t;
    cin >> t;
    while (t--) {
        int n, ans = 0;
        cin >> n;
        priority_queue<int,vector<int>, greater<int>>q;
        for(int i = 0; i < n; i ++) {
            int x;
            cin >> x;
            q.push(x);
        }
        while(q.size() != 1)
        {
            int t1,t2;
            t1 = q.top();q.pop();
            t2 = q.top();q.pop();
            ans += t1 + t2; //以和的形式被累加n次，即乘以它的深度n（除叶子节点外所有节点值的和）
            q.push(t1 + t2);
        }	
        cout << ans << endl;
    }
	return 0;
}

Pre-Post-erous!

We are all familiar with pre-order, in-order and post-order traversals of binary trees. A common problem in data structure classes is to find the pre-order traversal of a binary tree when given the in-order and post-order traversals. Alternatively, you can find the post-order traversal when given the in-order and pre-order. However, in general you cannot determine the in-order traversal of a tree when given its pre-order and post-order traversals. Consider the four binary trees below:

	a   a   a   a
	/   /     \   \
       b   b       b   b
      /     \     /     \
     c       c   c       c

All of these trees have the same pre-order and post-order traversals. This phenomenon is not restricted to binary trees, but holds for general m-ary trees as well.

输入：

  • Input will consist of multiple problem instances. Each instance will consist of a line of the form
  • m s1 s2
  • indicating that the trees are m-ary trees, s1 is the pre-order traversal and s2 is the post-order traversal.All traversal strings will consist of lowercase alphabetic characters. For all input instances, 1 <= m <= 20 and the length of s1 and s2 will be between 1 and 26 inclusive. If the length of s1 is k (which is the same as the length of s2, of course), the first k letters of the alphabet will be used in the strings. An input line of 0 will terminate the input.

输出：

  • For each problem instance, you should output one line containing the number of possible trees which would result in the pre-order and post-order traversals for the instance. All output values will be within the range of a 32-bit signed integer. For each problem instance, you are guaranteed that there is at least one tree with the given pre-order and post-order traversals.

样例输入：

2 abc cba
2 abc bca
10 abc bca
13 abejkcfghid jkebfghicda
0

样例输出：

4
1
45
207352860

题解

题目的大致意思是给定先序和后序遍历，要求输出可能的m叉树的数目。
先看先序序列，得出二叉树的根节点（第一个元素），再看第二个元素，是第一个子树（最多有m个子树，这里我最开始直接当作二叉树的左子树处理了，得到了错误的结果）的根节点；
而后序序列中左子树的根节点的位置即为左子树结束的位置，即先序序列的第二个元素在后序序列中的位置及其之前的部分为第一个子树；在这之后的序列里重复上述找子树根节点的过程，即可找到所有子树，记数目为k。 每棵子树内部，可用递归求解可能的情况树，而根节点与子树之间，有C_m k中排列的可能（m个分叉中选k个分叉作为子树不为空的分叉）。故实现思路为递归与组合计数结合。
要另外注意的是，如果用阶乘求解组合数，会涉及整型溢出的问题，很难发现，所以最好使用递推数组的方法，借助组合数的数学性质：

C_m k = C_m-1 k-1 + C_m-1 k

此外，因为用多了string类型导致忘了char*如何初始化和输入：

char s1[N];  scanf("%s", s1);

strlen,strcat等函数接受的是char*类型的参数。

代码实现

//Pre-Post-erous!
#include<iostream>
#include<cstring>
using namespace std;

int Comb[21][21];
void calComb()
{
    Comb[0][0] = 1;
    for(int i = 1; i <= 20; ++i)
    {
        Comb[i][0] = 1;
        Comb[i][i] = 1;
        for(int j = 1; j < i; ++j)
        {
            Comb[i][j] = Comb[i-1][j-1] + Comb[i-1][j];
        }
    }
}

int count(char* s1, char* s2, int len, int m) {
    int i = 1;
    int j = 0;
    int branch = 0;
    int answer = 1;
    while (i < len) {
        for (; j < len; j ++) {
            if (s1[i] == s2[j]) {
                branch ++;
                answer *= count(s1+i, s2+i-1, j-i+2, m);
                i = j+2;
                break;
            }
        }
    }
    answer *= Comb[m][branch];
    return answer;
}

int main()
{
    int m;
    calComb();
    while (cin >> m && m != 0) {
        char s1[27], s2[27];
        scanf("%s", s1);
        scanf("%s", s2);
        cout << count(s1, s2, strlen(s1), m) << endl;
    }
	return 0;
}

由中根序列和后根序列重建二叉树

我们知道如何按照三种深度优先次序来周游一棵二叉树，来得到中根序列、前根序列和后根序列。反过来，如果给定二叉树的中根序列和后根序列，或者给定中根序列和前根序列，可以重建一二叉树。本题输入一棵二叉树的中根序列和后根序列，要求在内存中重建二叉树，最后输出这棵二叉树的前根序列。

输入:

  • 两行。第一行是二叉树的中根序列，第二行是后根序列。
  • 每个数字表示的结点之间用空格隔开。结点数字范围0～65535。暂不必考虑不合理的输入数据。

输出:

  • 一行。由输入中的中根序列和后根序列重建的二叉树的前根序列。
  • 每个数字表示的结点之间用空格隔开。

样例输入：

9 5 32 67
9 32 67 5

样例输出：

5 9 67 32

题解

题目要求重建二叉树，但我觉得直接用递归很方便，所以没有在内存里建二叉树，下面讲一下递归的做法。
先由后根序列找出根节点（最后一个元素）；再在中根序列中找出根节点的位置，其左边为左子树，右边为右子树；接着对左右分别递归即可。
递归的结束条件是序列长度为0，要注意本题的输入为两行数据，且不知道元素个数，需要用getline和istringstream输入，vector储存：

#include <sstream>
getline(cin, line); //将第一行作为字符串读入line
istringstream medStream(line); //从line中读入数字元素
int num;
while (medStream >> num) {
    med.push_back(num); //存入vector
}

这在递归时要求用iterator的形式递归：

getPre(vector<int>(med.begin() + i + 1, med.end()),
		vector<int>(post.begin() + i, post.end() - 1),
   			len - i - 1);

代码实现

//由中根序列和后根序列重建二叉树
#include <iostream>
#include <cstring>
#include <queue>
#include <vector>
#include <sstream>
using namespace std;
#define MAX 100 
void getPre(const vector<int>& med, const vector<int>& post, int len) {
    if (len == 0) return;
    int root = post[len - 1];
    cout << root << ' ';
    if (len == 0) return;
    int i = 0;
    for (; i < len && med[i] != root; i++) {}
    getPre(med, post, i);
    getPre(vector<int>(med.begin() + i + 1, med.end()), vector<int>(post.begin() + i, post.end() - 1), len - i - 1);
}

int main()
{
    vector<int> med, post;
    string line;
    // 读取中序遍历数组
    getline(cin, line);
    istringstream medStream(line);
    int num;
    while (medStream >> num) {
        med.push_back(num);
    }
    // 读取后序遍历数组
    getline(cin, line);
    istringstream postStream(line);
    while (postStream >> num) {
        post.push_back(num);
    }

    int len = med.size();
    getPre(med, post, len);
    cout << endl;
	return 0;
}
